How to build the Windows Executable.

1) Install NSIS (Nullsoft Scriptable Install System) on a Windows PC. 
2) Start NSIS and click "Compile NSI scripts"
3) Open the jpoexe.nsi script (It immediately compiles the launcher)
4) Open the jpoinstaller.nsi script (It immediately creates the Jpo-Installer.exe program)

See http://sourceforge.net/projects/nsis
