Developing JPO with the Command Line

Ensure you have everything set up:

  • Have you downloaded a JDK? Oracle Java SDK
  • Have you got Gradle? https://gradle.org/
  • Have you got Git? https://git-scm.com/

git clone https://github.com/richardeigenmann/JPO.git
cd JPO

gradle test

gradle run
